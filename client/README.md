# Portfolio Frontend (React + Vite)

This is the frontend for the personal portfolio project.

## Run Locally

```bash
npm install
npm run dev
```

Default local URL: `http://localhost:5173`

## Environment

Create `client/.env`:

```env
VITE_API_BASE_URL=http://localhost:5000
```

## Build

```bash
npm run build
npm run preview
```

## Main Features

- Home, About, Projects, Contact sections
- Smooth scrolling + active nav section highlight
- Project modal with tech stack and external links
- Screenshot viewers for Weather, Clinic, and Operation Scheduler projects
- Contact form with client-side validation and backend submission
- Resume open/download buttons
